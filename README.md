AxiomaNode — модель диагностики пробелов в понимании квадратных уравнений
1. Проблема

В школьной математике оценка фиксирует результат (правильно/неправильно), но не выявляет процесс мышления, из-за чего ученики с разными типами непонимания получают одинаковую обратную связь.

В результате:

ученики механически заучивают формулы;

пробелы накапливаются;

ускорение темпа (классы A/B/C) усиливает разрыв.

2. Основная гипотеза

Большинство ошибок при решении квадратных уравнений связано не с вычислениями, а с разрывами между этапами мышления:

между смыслом и формулой,

между формулой и графиком,

между анализом задачи и выбором метода.

Если выявить конкретный разрыв, а не оценивать ученика числом, обучение становится эффективнее и психологически безопаснее.

3. Цепочка мышления при решении квадратных уравнений

Решение квадратного уравнения требует прохождения следующих этапов:

Интерпретация
Понимание, что уравнение описывает зависимость, а не просто выражение.

Репрезентация
Умение переходить между:

формулой

графиком

словесным описанием

Анализ
Способность рассуждать о числе корней и условиях их существования до вычислений.

Решение
Корректное применение алгоритмов (формулы, преобразования).

Проверка и осмысление
Оценка разумности ответа и его связи с исходной задачей.

4. Типы разрывов (ошибок)
Тип разрыва	Описание	Поведенческий признак
Формальный	Знание формул без понимания смысла	Начинает с подстановки, теряется в нестандартной задаче
Репрезентационный	Не может связать формулу с графиком или смыслом	Не понимает, что корень — это точка пересечения
Логический	Не видит условий существования решений	Не может объяснить, почему корней нет
Стратегический	Выбирает неверный метод решения	Использует формулу там, где нужен анализ
5. Принцип диагностики

Каждое задание проверяет один переход между этапами мышления, а не общий уровень сложности.

Результат диагностики — профиль понимания, а не числовая оценка.

6. Цель первой версии

Проверить гипотезу:

Диагностика по типам разрывов позволяет точнее выявлять причины ошибок и повышает понимание темы.
